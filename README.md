# 🚀 Quick Service Finder

## 📌 Overview
Quick Service Finder is a smart web-based platform that helps users find and book local service providers such as electricians, plumbers, tutors, and mechanics quickly and easily.

---

## 🎯 Features
- 🔐 User Login & Registration
- 🔍 Search for services
- 📅 Booking system (date & time selection)
- 📊 Admin dashboard
- ➕ Add/Delete service providers
- ⭐ Ratings & pricing display

---

## 🛠️ Tech Stack
- Frontend: React / HTML, CSS, JavaScript
- Backend: Python (Flask)
- Database: SQLite

## ⚙️ Installation & Setup

### 🔹 Backend Setup
cd backend  
python -m venv venv  
venv\Scripts\activate  
pip install flask flask-cors flask-sqlalchemy werkzeug PyJWT  
python app.py  

---

### 🔹 Frontend Setup
cd frontend  
npm install  
npm run dev  

---

## 🌐 Usage
1. Open browser and go to: http://localhost:5173  
2. Register/Login  
3. Search for services  
4. Book services
